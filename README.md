# 🌾 Rainfall Impact on Agricultural Productivity Using Power BI

## 📌 Project Overview

This project analyzes the impact of rainfall on agricultural productivity using Power BI. The dashboard provides interactive visualizations to understand rainfall trends, rice production, and agricultural productivity across different states.

---

## 🎯 Objectives

- Analyze rainfall trends over the years.
- Compare rice production across different states.
- Study the relationship between rainfall and rice yield.
- Provide actionable insights for agricultural planning.

---

## 🛠 Tools Used

- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

---

## 📊 Dashboard Features

### Page 1 – Rainfall Analysis Dashboard

- Total Rainfall KPI
- Average Rainfall KPI
- Total Rice Production KPI
- Average Rice Yield KPI
- Rainfall Trend Line Chart
- Rice Production by State
- Interactive Slicers

### Page 2 – Agricultural Productivity Dashboard

- Scatter Chart
- State-wise Production Map
- Key Insights
- Recommendations
- Conclusion

---

## 📈 Key Insights

- Rainfall varies significantly across years.
- Higher rainfall generally supports better rice production.
- Agricultural productivity differs across states.
- Proper irrigation can improve crop yield.

---

## 💡 Recommendations

- Improve irrigation systems.
- Promote rainwater harvesting.
- Encourage climate-smart farming.
- Use rainfall forecasting for crop planning.
